LIBRARY MANAGEMENT SYSTEMThe Library Management System is a professional, menu-driven command-line application developed in C++ using the Dev-C++ IDE. It simplifies daily library administrative tasks by replacing manual paperwork with digital automation, ensuring accurate data tracking for books, members, fines, and rules.SYSTEM FEATURESBook Management ModuleAdd Books: Register new books with their ID, Title, Author, and Publication Year.Search Books: Instantly look up books using their unique Book ID.Check Availability: Verify if a book is currently available in the library or issued to a member.Issue Books: Assign available books to registered members using their Member ID.Remove Books: Delete outdated or damaged book records from the system.Low Inventory Warnings: Automatically alerts administrators when library stock drops to two books or fewer.Member Management ModuleRegister Members: Create new profiles with a Member ID, Name, Roll Number, and Department.Search Profiles: Instantly retrieve member details and view their list of currently issued books.Remove Members: Cleanly delete member entries from the registry.Capacity Notifications: Displays alerts when membership levels are low or reaching full storage limits.Fine and Rules SystemDynamic Fine Calculator: Simulates and displays late-fee accumulation over time for overdue materials.Library Policies: Direct access to operating hours, borrowing limits, and official university rights.SYSTEM ARCHITECTUREThe application is structured logically with clean conditional flows, loops, and custom functions:Main Menu: The central hub redirecting users to various operational modules.Static Storage: Configured using standard array structures for fast, lightweight database tracking.Admin Security: Restricts system access via an administrative credentials portal.
